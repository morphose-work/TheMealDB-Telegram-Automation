# TheMealDB Telegram Automation

С помощью этой автоматизации на n8n в 12:20 вы будете получать 3 случайных рецепта

### 🚀Как его скачать:
1) Зарегестрируйтесь на n8n
2) Зайдите в ваш workflow
3) Нажми на 3 точки сверху правом углу
4) Нажми import from file и выбери файл с автоматизацией
5) Впиши туда свои данные

### Список данных:
1) ChatID:
 - Зайдите в Телеграм и найдите бота @Getmyid_bot
 - Нажми /start
 - Строчка с Your ID и есть ваш ChatID
2) TelegramBot API:
 - Зайдите в Телеграм и найдите бота @BotFather
 - Создайте бота и скопируйте его ключ
